## ğŸ”§ Customization

### Update Personal Information
1. Edit `index.html` to update content
2. Replace placeholder links with your actual project/credential URLs
3. Add your profile image and project screenshots
4. Update meta tags with your domain

### Change Colors
Edit the CSS custom properties in `:root`:
```css
:root {
    --accent-1: #5b7cfa;  /* Primary accent */
    --accent-2: #8c52ff;  /* Secondary accent */
    --accent-3: #00d4ff;  /* Tertiary accent */
}
```

### Add New Projects
Copy an existing `<article class="project-card">` block and update the content.
